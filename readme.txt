== Description ==

Luzuk Slider is built using WordPress best practices both on the front and the back end. It is the most advanced responsive and WordPress slider plugin. It works smoothly on desktop and devices. It's works with any theme, including the default WordPress themes. You can add there button with links. You can add button label with font awesome icon also. You may change 'Transition Effect' for slider. You can add various feilds like :  Title (Slider main label), Description (Slider other label), button, button links.

== Installation ==

= Minimum Requirements =

* PHP version 5.4 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 7.0 or greater
* MySQL version 5.6 or greater
* WordPress Memory limit of 64 MB or greater (128 MB or higher is preferred)

= Automatic installation (easiest way) =

To do an automatic install of "Luzuk Slider", log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Luzuk Slider" and click Search Plugins. Once you have found it you can install it by simply clicking "Install Now".

= Manual installation =

**Uploading in WordPress Dashboard**

1. Download "luzuk-slider.zip"
2. Navigate to the 'Add New' in the plugins dashboard
3. Navigate to the 'Upload' area
4. Select "luzuk-slider.zip" from your computer
5. Click 'Install Now'
6. Activate the plugin in the Plugin dashboard

**FTP install** 

1. Download "luzuk-slider.zip"
2. Extract the "luzuk-slider" directory to your computer
3. Upload the "luzuk-slider" directory to the "/wp-content/plugins/" directory
4. Activate the plugin in the Plugin dashboard.
